# Next.js Project Setup

## Installation

Clone the repository and install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

## Running the Development Server

Start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Open in Browser

Visit [http://localhost:3000](http://localhost:3000) to view the application.
